# Trim Reaper - Tree Trimming & Brush Removal Landing Page

Professional tree trimming and brush removal services landing page for Greater Los Angeles area. Built with Next.js 16 and React 19.

## About Trim Reaper

Family-owned tree service company serving Silver Lake, Echo Park, Hollywood, Highland Park, Pasadena, Glendale, and surrounding areas. We specialize in:

- Tree Trimming (Crown reduction, shaping, pruning)
- Brush Removal & Fire Hazard Clearance
- Emergency Storm Damage Service
- Complete Haul Away & Cleanup

**Contact:** (213) 555-0123

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Tech Stack

- **Framework:** Next.js 16.1.0
- **React:** 19.2.3
- **Styling:** Tailwind CSS 4 + Custom CSS
- **TypeScript:** 5
- **Icons:** Material Icons, Font Awesome, Bootstrap Icons

## Project Structure

```
├── app/
│   ├── layout.tsx       # Root layout with metadata & scripts
│   ├── page.tsx         # Main landing page
│   ├── globals.css      # Tailwind imports
│   └── styles.css       # Custom styling
├── public/
│   └── images/          # Tree service images & logos
├── Configuration files
```
